This is a list of keyboard shortcut I find useful when working with Mac:
Shift + Command + SquareBracketKey 		Switch browser tab
Command + W 					Close current window
Command + Q					Close all window
Command + T					Open new web browser tab
Command + C 					Copy
Command + X					Cut
Command + V					Paste
Command + F					Find
Command + Shift + 3				Take screenshot of entire screen
Command + Shift + 4 				Take screenshot
